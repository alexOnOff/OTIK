
Формат файла: .nkvd

Хедер
1. Сигнатура 4 байта - { 0x6e , 0x6b, 0x76, 0x64 }
2. Версия 1 байт
3. Подверсия 1 байт
4. Код шифрования 1 байт
5. Код еще чего-то на всякий случай 1 байт

Бади
Для каждого файла
1. Длина названия файла (int) 4 байта - lenFileName
2. Название файла (lenFileName) байт
3. Длина содержания файла 4 байта (int, надеюсь хватит) - lenFile
4. Содержимое файла (lenFile) байт
